Thopaka

E-commerce de Antigüedades y Artesanías

Thopaka es un pequeño negocio dedicado a la producción y venta de artesanías, así como también la compra local y extranjera de antigüedades para ser ofrecidas al público.

Esta microempresa tiene un limitante con el acceso a información de sus productos por parte de los usuarios para un proceso de compra eficiente.

Por eso decidí optar por un diseño sencillo y minimalista, manteniendo la gama de colores de Thopaka y mostrando los productos y precios desde un principio. Este aplicativo es un medio para poder reservar un producto y luego ver la información de contacto con la empresa, y así los usuarios puedan visitar la tienda física; o incluso comunicarse para plantear un posible delivery, si el usuario lo requiere.

SoyTuPata

Red social para temas de salud de mascotas

Este fue el primer proyecto que estuvo mucho más marcado en el aspecto de UX Design. Desde investigación, entrevistas, análisis, ideación, hasta prototipado y testings, además de programar la red social. Fue muy enriquecedor en mi aprendizaje y cementó mucho más mi gusto por UX Design.

Esta red social tiene como prioridad facilitar la comunicación de los usuarios para que puedan encontrar la información y el apoyo que necesitan. En esta aplicación se puede hacer publicaciones en el muro, editarlas o eliminarlas, especificar la categoría a la que pertenecen y poner “Me gusta” a una publicación.

El diseño del Log in se pensó como una idea de dinamismo. Los colores que usamos son el verde y el blanco, colores orgánicos, que denotan salud y bienestar. Proporciona una sensación de tranquilidad y seguridad.

Como UX Designer elaboré el prototipo móvil y desktop.

Data Dashboard Laboratoria

Herramienta para visualizar y manejar información

Los principales usuarios del producto son las Training Managers (TMs) de Laboratoria, quienes no tienen una interfaz para ver y manejar la información sobre el avance de las estudiantes.

Después de hacer el research necesario, mi compañera de proyecto y yo analizamos la información y diseñamos una solución inspirada en el LMS de Laboratoria. Utilizamos sketches y prototipos. Luego, maquetamos en html y css, y por último se desarrolló con JavaScript todas las funciones para poder filtrar, buscar y visualizar la data brindada por el API de Laboratoria.
Este proyecto fue muy retador.

Registro de Usuarios

Aplicativo para registrar usuarios en empresa coworking

El usuario principal es el administrador que necesita registrar y visualizar la data de los visitantes de su empresa coworking.

Al principio, con mi equipo habíamos pensado que habían dos usuarios principales. Uno era el administrador que necesita visualizar los datos de los visitantes, y el otro era el mismo visitante.

Esta separación de usuarios era porque el aplicativo permitiría que se pueda registrar por sí mismo, y que la persona en recepción pueda recibir la data y manejarla. Pero después de realizar las entrevistas, especialmente a la persona que se encuentra en la recepción, sacamos la conclusión de que el aplicativo estaba mejor enfocado para que lo utilice sólo el administrador.

Cipher On!

Aplicativo para cifrar y descifrar mensajes

Este fue el primer proyecto que realicé como estudiante en Laboratoria. Fue todo un reto porque era la primera vez que realmente programaba.

Esta aplicación web es capaz de cifrar y descifrar mensajes con un número de desplazamiento establecido por el usuario, para que pueda comunicarse con alguien sin que una tercera persona sepa lo que está diciendo.
